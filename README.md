# LeadForge — AI B2B Lead Intelligence Platform

> Generate enriched B2B leads for any niche in seconds — 
> powered by Google Gemini AI. No signup. No subscription. Free.

🔗 https://leadforgeproject.vercel.app/
---

## 🧠 What It Does

LeadForge takes a target niche and location as input and instantly 
generates enriched B2B leads complete with:

- Company name + website
- Decision maker name + job title
- Professional email + phone number
- Company size
- Lead quality score (1–100)
- Relevant tags
- Pain point specific to that business
- Personalized cold outreach hook

Export everything to CSV in one click — paste directly into 
Google Sheets, Airtable, or any CRM.

---

## ⚡ Features

- 🎯 15+ industry niches + custom niche input
- 📍 Location-based lead filtering
- 🤖 Gemini 1.5 Flash API for real-time AI enrichment
- 📊 Color-coded quality scores (green / blue / red)
- 📥 One-click CSV export
- 💻 Premium dark glassmorphism UI
- ⚡ Zero backend — fully serverless
- 🆓 100% free — no login required
